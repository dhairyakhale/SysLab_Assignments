GENERAL INSTRUCTIONS:
1. Insert your input in input.txt
2. The input should be in appropriate format
3. The output will be relayed in output.txt

INPUT FORMAT:
1. Line 1 contains the amount of single, double and triple occupancy rooms respectively, separated by spaces.
2. Line 2 contains an integer which is the number of queries in the next line.
3. Line 3 contains the queries 'S','D','T', separated by spaces.
